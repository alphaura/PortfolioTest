Portfolio - 2.0

Check out how I came up with creating my portfolio version 2.0, from the design aspect all the way to having it coded and finalized.

HTML & CSS

Having recently earned the Adobe Certified Professional in Web Authoring Using Adobe Dreamweaver certification through ICE Malta, I wanted to re-vamp my portfolio site and showcase my skills as a web designer and a web developer by creating the initial design using a design tool which in this case was Figma, and then develop the actual design into a working responsive website.

As one will be able to notice, some design areas aren't like how the portfolio is looking as I haven't yet found the answer to how I can create what I want to, however, I found alternatives that are working just fine, in order to still have my portfolio responsive and looking slick. While it's not visible in the photo, I have also used the "comment" tool in Figma to provide more context to the design and of how I want it to work, should the content has some sort of functionality to it.

If I had to choose my most favourite part of the site so far, I would go with the homepage animation. With a good amount of research and some tweaks in the prompts I was giving to ChatGPT, I managed to find a way on how I can make the "languages and tools" section move from right to left automatically on a mobile device. While it's nothing extraordinary, I thought of making such animation in order not to over stack the display with all the content, and therefore, I made use the "invisible space", or so I call it like that haha.

I'm always seeking more knowledge, especially JavaScript and/or similar languages which by the time of this writing, I had enrolled myself in a Scrimba course of JavaScript, in order to improve my portfolio itself and also the future projects that I will be working on. If you have any suggestions, tips or opinions, feel free to share them with me, I'd be happy to look into them and discuss them with you!
