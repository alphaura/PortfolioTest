WORK

Responsive Site

This was the final project of a 21-day course I completed, which required the creation of a fully responsive site following a design provided by the educator.

HTML & CSS

Live Demo

The main goal of this project was to develop the final design provided into an actual fully responsive site. I was provided with the full specs of the font-sizing, assets, how it should look like in desktop, medium and mobile-size devices.

In this project I have mostly used `flex-box` and `media-queries` in order to play with the content resizing and responsiveness on the different device sizes, due to mainly being what I have mostly learned throughout this course, as it was based on the understanding of the importance of responsive design, mobile-first mindset, flex-box, and media-queries.
