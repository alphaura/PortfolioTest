Four Card Section

NAME OF PROJECT

Four Card Section

LANGUAGES USED

HTML & CSS

DESCRIPTION

Re-creating a four card responsive section using flex-box.

VIEW CHALLENGE ↗ VIEW PROJECT ↗

The Process

The best thing about this challenge, is that it had a small responsive requirement which I enjoy working on. I'm getting used to how flex-box works which has helped me find out another way in using flex-box. Upon seeing the design, I was already seeing where I was going to use it, but wasn't so sure how I will do it.

Nonetheless, like any other responsive design, I had to work mobile-first, which really helps when it comes to making the rest of the project responsive to the design you want.

The Design

The design was already provided within the FrontEnd Mentor challenge itself, as the aim of these challenges is to replicate the design as much as possible following the information provided upon starting the challenge as images and a READme file are provided as a guide.

The Challenge

After I finished the mobile design, I went on to do the finishing touches for the desktop design, but I then noticed that the HTML structure for the cards wasn't going to make the design work (meaning having the middle section with two cards on top of each other), so I simply put the middle cards in a class with the name of `container` which then made it work splendidly, and found out a new way to use flex-box to make different layouts.
