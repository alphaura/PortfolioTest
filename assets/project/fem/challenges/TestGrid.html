Testimonial Grid Section

NAME OF PROJECT

Testimonial Grid Section

LANGUAGES USED

HTML & CSS

DESCRIPTION

Re-creating a Testimonial grid using Grid and specific area naming for optimal positioning.

VIEW CHALLENGE ↗ VIEW PROJECT ↗

The Process

This was the first challenge I got to do, where I was able to solely focus on applying the Grid knowledge I had recently got, after completing the unusual and fun course of Grid Critters by Dave Geddes. The goal of this challenge was to re-create the testimonial grid design provided while using Grid in order to manage flexible and specific positioning of the cards.

The Design

The design was already provided within the FrontEnd Mentor challenge itself, as the aim of these challenges is to replicate the design as much as possible following the information provided upon starting the challenge as images and a READme file are provided as a guide.

The Challenge

I had a few challenges in the beginning as I was implementing `grid` to the wrong tags and not directly with the parent, so I took some time to just understand what the issue was, what I wanted to do, and then searched for the answer

After identifying the issue, I started from fresh again, structured the HTML in a better way that I can understand it, fixed the issue, and understood more how Grid can be used for basic functionality such as doing a testimonial section. To place the cards in their places, I used `grid-area`, as I feel it makes my life easier to create the areas of where I want the blocks to go, and then it's even easier to place the cards in these blocks.
