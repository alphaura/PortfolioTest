Testimonial Grid Section

Project using CSS Grid in order to make a responsive testimonial section

HTML & CSS

View Challenge

This was the first actual challenge I got to do, where I was able to solely focus on applying the Grid knowledge I had recently got, after completing the fun "course" of Grid Critters by Dave Geddes. I had a few challenges in the beginning as I was implementing `grid` to the wrong tags and not directly with the parent, so I took some time to just understand what the issue was, what I wanted to do, and then searched for the answer, which I found and helped me fix it.

After identifying the issue, I started from fresh again, structured HTML in a better way that I can understand it, fixed the issue, and understood more how Grid can be used for basic functionality such as doing a testimonial section. To place the cards in their places, I used `grid-area`, as I feel it makes my life easier to create the areas of where I want the blocks to go, and then it's even easier to place the cards in these blocks.
