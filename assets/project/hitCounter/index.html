Hit Counter

First project with the focus on JavaScript implementation, of a working hit counter for paintball

HTML, CSS & JavaScript

Live Demo

My very first JavaScript focused project. In a previous course that I have done, we only brushed on JavaScript which was enough to get a decent grasp of some of the basic fundamentals, however, I wanted to delve into this language, in order to complete the fundamental stack for web development.

This project was created as part of the Scrimba course that I'm doing. While the JavaScript itself is very basic it was a very good starter as it allowed me to understand a bit more how this language is connected with HTML. Having responsiveness on top of mind, I decided to restructure the HTML provided in the course, to make the whole project responsive and played a bit with the good old friend flex-box.
